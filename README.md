# iOS News Reader App
A sample iOS application developed in Swift that fetches news articles from a REST API and displays them in a list and grid format.
This project demonstrates clean architecture, MVVM pattern, pagination, Core Data, and API integration.

--------------------------------------------------------------------------------------------------------------------------------------------

# Features
- Fetch news articles from API
- List View
- Search functionality
- Redirect to Safari browser for Article details
- Feature for bookmark
- Image loading and caching
- CoreData offline storage
- Error handling\strokec3

--------------------------------------------------------------------------------------------------------------------------------------------

# Architecture
This project follows MVVM (Model View ViewModel) architecture.
Model: Data layer (API response models)
View: UI components
ViewModel: Business logic and API communication

--------------------------------------------------------------------------------------------------------------------------------------------

# Technologies Used
- Swift
- UIKit
- URLSession
- CoreData
- MVVM Architecture

--------------------------------------------------------------------------------------------------------------------------------------------
